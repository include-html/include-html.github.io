Install

No NPM package! This Web Component is about Less Code and Using less dependencies.

  • Copy the element-include-html.js file to your project.
  • Steal the code and work it into your own BaseClass
  • Include the minified JS code in a <script> tag in your (main) HTML file.
  • For testing purposes load it from this GitHub repo:
    <script src="https://include-html.github.io/element-include-html.js"></script>
